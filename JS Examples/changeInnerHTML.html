My First JavaScript

JavaScript can change the content of an HTML element. It can be done by assigning an element an id and use that id to change the inner content of the element as we can see two examples here: one changing the element by a p tag and the other by an h1 tag.

This is a demonstration that will be modified by JavaScript when the user clicks on the Click Me! button.

Note that we used an id to identify an element for programming in JavaScript as well as controlling the style in CSS as we did last time.

In the HTML DOM object model, the document object represents your web page. Within that object, we can identify elements.

You can set any value that is valid in HTML as the value of innerHTML enclosed in double quotes. If you must have a quoted string inside another string, use a single quote pair inside a double quote pair as it is done in Line 1 above. Or, alternatively, a double quote pair inside a single quote pair.
