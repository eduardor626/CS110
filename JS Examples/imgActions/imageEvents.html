These examples demostrate Javascript events on images (but it can be any element). See this link for other JS events W3Schools .

Example 1

The function bigImg() is triggered when the user moves the mouse pointer over the image.

The function normalImg() is triggered when the mouse pointer is moved out of the image.

Example 2

This example uses the addEventListener() method to attach a "mouseover" and "mouseout" event to a img element.

Example 3

This example uses the addEventListener() method to attach a "onclick" to a img element.
